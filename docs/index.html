Q : 如何标记已持有角色?
                               

显示灰色的图示为未持有状态，只要点击一次就会切换成标记持有状态并显示为红色。如在红色的图示上点击，将切换为未持有状态。

                               

如在灰色的★５的图示上点击，将自动标记☆４为持有。

                               

如在红色的☆４的图示(或最低星)上点击，将认为没有此角色，★５图示也会自动取消标记。

                           
                       
                       
                           
                               
Q : 如何快速找到角色?
                               

角色表的角色数量很多的关系，为加快选择特定角色的速度，在屏幕下方会有以下三个筛选按键。

                               

【武器分类】【属性分类】【重置筛选器】

                               

点选一次条件会进行筛选，再次点选相同的条件会取消该筛选条件。

                           
                       
                       
                           
                               
Q : 如何选择产生的表格?
                               

纯文字不包含表格，复制后可贴上到任何地方。

Q : 如何標記已持有角色?

顯示灰色的圖示為未持有狀態，只要點擊一次就會切換成標記持有狀態並顯示為紅色。如在紅色的圖示上點擊，將切換為未持有狀態。

如在灰色的★５的圖示上點擊，將自動標記☆４為持有。

如在紅色的☆４的圖示(或最低星)上點擊，將認為沒有此角色，★５圖示也會自動取消標記。

Q : 如何快速找到角色?

角色表的角色數量很多的關係，為加快選擇特定角色的速度，在屏幕下方會有以下三個篩選按鍵。

【武器分類】【屬性分類】【重置篩選器】

點選一次條件會進行篩選，再次點選相同的條件會取消該篩選條件。
